## Mass FTP mounter by ro0fy

This is a simple script for mass ftp servers mounting.

## REQUIREMENTS<br>
* Curlftpfs program<br>
* *nix system<br>
* python3<br>

## INSTALLATION & RUNNING<br>

Just start the main script
`python3 main.py`<br>
